Process Proces
⚙

AI-First Development

Every project is built on a 100% human idea, developed in collaboration with new AI tools from Claude (Anthropic). AI handles architecture, code generation, and data processing — human brings the vision.

Rozwój z AI

Każdy projekt opiera się na 100% ludzkiej idei, rozwijanej we współpracy z nowymi narzędziami AI od Claude (Anthropic). AI odpowiada za architekturę, generowanie kodu i przetwarzanie danych — człowiek za wizję.

◆

Self-Contained Apps

Pure HTML/CSS/JS — no frameworks, no build steps, no dependencies. Each project is a single deployable artifact that runs anywhere.

Samodzielne aplikacje

Czysty HTML/CSS/JS — bez frameworków, bez budowania, bez zależności. Każdy projekt to pojedynczy artefakt, który działa wszędzie.

+

Growing Collection

New projects are added regularly. Each one explores a different domain — data journalism, education, investigative tools, interactive storytelling.

Rosnąca kolekcja

Nowe projekty dodawane regularnie. Każdy eksploruje inną dziedzinę — dziennikarstwo danych, edukację, narzędzia śledcze, interaktywne opowiadanie historii.
